🌲 Forest Drive 3D Resume Setup

⚠️ Important: The 3D Forest Scene runs on a development server that needs to be started before you can use it.

🚀 Super Easy Setup (Recommended)

1. Run the startup script in your project folder:
start-forest-drive.bat
2. Keep the server window open (it manages everything automatically)
3. Use the dashboard normally - servers start automatically!
4. Dashboard: http://localhost:3000

🎯 How It Works

The new system uses a manager server that:

  • ✅ Automatically starts the 3D scene server when needed
  • ✅ Restarts it if it crashes
  • ✅ Provides a seamless experience
  • ✅ No manual server management required

🔧 Troubleshooting

Common Issues:

  • Port 8080 already in use: Make sure no other server is running on port 8080
  • npx not found: Make sure Node.js is installed on your system
  • Permission denied: Try running as Administrator
  • Module not found: Run npm install in the project directory first

Prerequisites:

  • Node.js (version 14 or higher)
  • npm (comes with Node.js)
  • Modern web browser
