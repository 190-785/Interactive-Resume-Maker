‚ùå Error: Whitelabel Error Page (404)

Root Cause: You're accessing the backend Spring Boot server (port 8080) instead of the frontend 3D scene server (port 3000).

Solution: The frontend 3D scene runs on port 3000, not 8080!

üöÄ Quick Fix Solution

Method 1: Use the Direct Starter (Recommended)

Step 1: Double-click the file: start-3d-resume.bat
Step 2: Wait for "webpack compiled successfully" message
Step 3: Your browser should automatically open to http://localhost:3000
Step 4: Keep the command window open while using the application

Method 2: Manual Command Line

Step 1: Open Command Prompt or PowerShell
Step 2: Copy and paste this command:
cd "F:\Interavtive_Resume\Interactive-Resume-Maker\FrontEnd\Forest_Drive" && npx webpack serve --config webpack.config.js --mode development --port 3000
Step 3: Press Enter and wait for the server to start
Step 4: Open your browser to: http://localhost:3000

üîß Common Issues & Solutions

Issue: "Node.js is not recognized"

Solution: Install Node.js from https://nodejs.org/

Issue: "Port 3000 is already in use"

Solution: Close any existing server or use a different port:

npx webpack serve --config webpack.config.js --mode development --port 3001

Issue: "Module not found" errors

Solution: Install dependencies:

cd "F:\Interavtive_Resume\Interactive-Resume-Maker\FrontEnd\Forest_Drive" && npm install
